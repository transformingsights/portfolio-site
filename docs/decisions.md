# Decisions

## Design Decisions
- Black & white base palette
- No gold accents
- Cosmic hero direction approved
- Grain/noise texture required

## Typography
- Cal Sans (headings)
- Afacad (body)
- Custom font files provided

## Interaction Decisions

### Scroll Mask
- Single text block
- Transparent text
- 3 background image transitions
- Black background always present

### Hero
- Orbiting labeled circles
- Animated paths
- Starfield background

## Layout
- Must match wireframes exactly
- No creative reinterpretation of structure
