# Context

## Background
User has already created full wireframes for:
- Homepage
- About page
- Multiple case study pages

The goal is to translate these wireframes into a fully interactive portfolio.

---

## Key Design Intentions

### Visual Identity
- Bold, modern, editorial
- Lowercase typography
- Clean but expressive
- Strong art direction

### Hero Inspiration
- Space / cosmic theme
- Black background
- Grain/noise texture
- Starfield
- Motion-based orbit system

---

## Interaction Details

### Orbiting Elements
- Circular paths
- Continuous motion
- Labels inside circles
- Should feel smooth and intentional

---

### Text Mask Effect
Inspired by:
- "VACATION" style masking references

Important clarification:
- NOT outline text
- NOT multiple text sections
- MUST be:
  - transparent text
  - images visible THROUGH letters
  - background stays black
  - images change on scroll

---

## Pain Points Identified
- Previous version did NOT match wireframes
- Typography was incorrect
- Hero aesthetic was wrong (used gold, which is rejected)
- Scroll mask behavior was misunderstood

---

## User Expectations
- High fidelity to design
- Strong visual execution
- Clean, modern interactions
- No generic portfolio feel
